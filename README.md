# _Resource Hub_

### _Resource Hub is a modern file-upload web platform built with React, Node.js, Webpack, and Supabase. It allows users to upload files, manage their dashboards, and recover forgotten passwords. Admins get an advanced control panel to manage users, assign admin roles, and restrict upload permissions._

## ‚öôÔ∏è Installation & Setup Requirements
1. _Clone the repository_
> _git clone https://github.com/your-repo/fresher-hub.git_
>
> _cd fresher-hub_

2. _Install dependencies_
> _npm install_
>
> _If you face dependency conflicts: npm install --legacy-peer-deps_

3. _Create .env file inside your root folder:_
> __VITE_SUPABASE_URL=your_supabase_url_here_
>
> _VITE_SUPABASE_ANON_KEY=your_publishable_key_here_
>
> _RESEND_API_KEY=your_resend_key_here_
>
> _FRONTEND_URL=http://localhost:3000_

4. _Start the development server_
> _npm run dev_

5. _Production build:_
> _npm run build_
